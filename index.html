什麼是 NemoClaw？

NVIDIA NemoClaw 是 NVIDIA 於 2026 年 3 月正式開源的企業級 AI Agent 框架，基於 OpenClaw 架構但針對企業環境深度優化。它內建嚴格的安全性、資料隱私和合規控制，解決企業部署自主 AI Agent 的主要痛點。

NemoClaw 提供從模型層到執行環境的完整堆疊，成為新世代 AI Agent 基礎設施標準。

快速開始

# 安裝 NemoClaw
npx clawhub@latest install nemoclaw

# 設定 API Key
export NEMOCLAW_API_KEY="your_key"

# 執行 Agent
nemoclaw run --agent my-agent
